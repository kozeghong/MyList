# MyList

MyList is a simple todolist app, build on Electron, React and Redux.

Including:

  * Electron
  * React
  * Redux
  * Webpack
  * ES6
  * CSS Modules
  * SCSS

## Screenshot

<img src="assets/screenshot.png" alt="Screenshot">

## Build & Run

```bash
npm run build:dist
```

```bash
npm start
```

## Dev

```bash
npm run build:dev
npm run dev-server
npm start
```
