# MyList

MyList is a simple todolist app, build on Electron, React and Redux.

Including:

  * Electron
  * React
  * Redux
  * Webpack
  * ES6
  * CSS Modules
  * SCSS
  * Yarn

## Screenshot

<img src="assets/screenshot.png" alt="Screenshot">

## Build & Run

```bash
yarn build:dist
```

```bash
yarn start
```

## Dev

```bash
yarn build:dev
yarn run dev-server
yarn start
```
